## 📝 技術實現細節

### 1. i18n 配置
- 使用 Vue I18n (Composition API 模式)
- 語言偏好儲存在 localStorage
- 自動從 localStorage 讀取初始語言
- 支援參數化翻譯（如 `{name}`, `{count}`, `{error}`）

### 2. Element Plus 整合
- 動態載入 Element Plus 語言包
- 切換語言後自動刷新頁面以應用新語言包
- 確保所有 UI 組件（對話框、表單、表格等）正確翻譯

### 3. 程式碼實現
- 模板使用 `$t()` 進行翻譯
- Script 使用 `t()` （從 useI18n 解構）
- 所有硬編碼的中文文字已替換為翻譯鍵值
- 支援帶參數的動態翻譯

## 🚀 使用指南

### 開發者

#### 添加新的翻譯鍵值：
1. 在 `src/i18n/locales/zh-TW.js` 添加中文
2. 在 `src/i18n/locales/en-US.js` 添加對應英文
3. 在組件中使用 `$t('key')` 或 `t('key')`

#### 帶參數的翻譯：
```javascript
// 語言包
deleteConfirm: '確定要刪除作業「{name}」嗎？'

// 使用
$t('activity.deleteConfirm', { name: activity.name })
```

### 使用者

#### 切換語言：
1. 點擊頁面右上角的定位圖標 📍
2. 從下拉選單選擇：
   - 繁體中文
   - English
3. 頁面將自動刷新並套用新語言
4. 語言偏好會自動儲存
