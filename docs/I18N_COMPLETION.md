## 💡 使用提示

### 開發者
- 新增功能時記得添加翻譯鍵值到兩個語言包
- 使用 `$t('key')` 或 `t('key')` 進行翻譯
- 帶參數的翻譯使用 `$t('key', { param: value })`

### 使用者
- 語言偏好會自動儲存
- 切換語言後頁面會自動刷新
- 所有主要功能完全支援雙語
