### 測試步驟

1. **清除緩存**
   ```bash
   # 刪除 Vite 緩存
   rmdir /s /q node_modules\.vite
   
   # 重啟開發伺服器
   npm run dev
   ```

2. **清除瀏覽器緩存**
   - 按 `Ctrl + Shift + R` 硬重新整理

3. **切換語言**
   - 點擊右上角定位圖標 📍
   - 選擇 "English"
