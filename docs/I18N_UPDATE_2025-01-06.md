## 🧪 測試建議

### 完整測試流程

1. **清除緩存**（重要！）
   ```bash
   # 刪除 Vite 緩存
   rmdir /s /q node_modules\.vite
   
   # 重啟開發伺服器
   npm run dev
   ```

2. **清除瀏覽器緩存**
   - 按 `Ctrl + Shift + Delete`
   - 或按 `Ctrl + Shift + R` 硬重新整理

3. **切換到英文**
   - 點擊右上角定位圖標 📍
   - 選擇 "English"

4. **測試作業活動管理**
   - 進入專案管理頁面
   - 點擊任一專案的 "Manage Activities"
   - 檢查表格所有欄位標題是否為英文
   - 點擊 "Add Activity" 按鈕
   - 檢查對話框標題：應顯示 "Add Activity"
   - 檢查所有區段標題：
     * Basic Information ✓
     * Normal Activity ✓
     * Crash Activity ✓
     * Activity Relationship ✓
   - 檢查所有表單欄位標籤
   - 檢查所有 placeholder 文字
   - 檢查按鈕：Cancel、Confirm
   - 點擊任一作業的編輯按鈕
   - 檢查對話框標題：應顯示 "Edit Activity"

5. **測試 CSV 匯入**
   - 點擊 "Import CSV" 按鈕
   - 檢查對話框內容是否完整翻譯

6. **測試圖表組件**
   - 進入投標最佳化頁面
   - 執行計算後查看結果
   - 如果沒有數據，應顯示 "No data to display"

7. **測試專案狀態**
   - 返回專案管理頁面
   - 檢查表格標題應顯示「專案狀態」（中文版）
   - 切換到英文，應顯示 "Status"
